# Graph source of truth is a shared serialized model

Houdini Clone uses a versioned serialized graph model as the durable source of truth for node graphs. The frontend may hold editable in-memory views and the native backend may evaluate graph snapshots, but saved projects, procedural assets, plugins, undo/redo, and web/desktop parity should all depend on the shared serialized model rather than runtime-specific graph state. Durable references, including reference input operator targets, are serialized by stable identities with readable graph and node paths as user-facing labels; missing references remain explicit serialized diagnostics rather than being silently rebound by name. Reference navigation is derived from the serialized graph model, not from viewer state or editor-only wires.
