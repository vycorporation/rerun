# Graph UX prototype uses SvelteKit and Python-driven Rerun

The first graph UX prototype uses SvelteKit and Svelte Flow for the graph, layer stack, parameter, asset, and orchestration UI, while Python is used to drive Rerun and create data examples during the spike. Notebook workflows are valuable for validation, but the product UX still needs dockable panels, project commands, graph editing, layer controls, and asset authoring outside the notebook.
