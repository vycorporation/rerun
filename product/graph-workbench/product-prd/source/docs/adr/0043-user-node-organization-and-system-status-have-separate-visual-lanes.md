# User node organization and system status have separate visual lanes

Houdini Clone lets users apply graph styling such as node colors, network box colors, network box labels, graph notes, and connection routing dots for organization, while the system owns badges and status indicators for stale, running, cached, warning, error, selected, current, layer-promoted, output-targeted, managed, adopted, unbound, and commented states. Custom graph styling must not encode or obscure system status.
