# Network boxes organize without changing data flow

Houdini Clone supports Houdini-like network boxes, graph notes, and connection routing dots for visual organization of exploratory node graphs. Network boxes can be named, labeled, colored, collapsed visually if useful, and used to group related graph layout items; moving a network box moves its contained layout items, while moving items into or out of a network box changes visual organization only. Network box labels annotate contents as organization metadata, while longer explanations belong in graph notes or node comments. Graph notes are plain-text sticky-note-style graph layout items that can be moved, resized, colored, minimized if useful, and included in network boxes; rich asset documentation or reports should live outside graph notes. Deleting a network box deletes only the box unless the user explicitly chooses to delete its contents. Connection routing dots can route, gather, or tidy connection paths. Pinned connection routing dots may persist without active connections as layout scaffolding. Collapsed network boxes are compact canvas presentation, not subgraphs, procedural assets, hidden containers, or evaluation boundaries. These organization affordances do not change data flow, typing, graph evaluation, layer bindings, output participation, or asset boundaries.
