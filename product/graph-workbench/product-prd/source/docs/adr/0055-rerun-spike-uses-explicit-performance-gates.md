# Rerun foundation decision uses explicit performance gates

The Rerun foundation decision must use explicit performance gates rather than subjective impressions. Point-cloud gates can follow the initial graph UX spike, but before Rerun is accepted as the long-term foundation it should test point-cloud tiers at 1M, 10M, and 100M points, plus a curve- and polygon-heavy benchmark dataset, and it only passes if navigation remains usable, node or layer metadata can be inspected, visibility can be toggled, and simple style and filter equivalents can be applied without the viewer becoming unusable.
