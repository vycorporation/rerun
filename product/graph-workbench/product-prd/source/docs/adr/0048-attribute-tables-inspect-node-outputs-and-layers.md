# Attribute tables inspect node outputs and layers

Houdini Clone v1 includes attribute table views for inspecting records and attributes from node outputs and layer views. Attribute tables support understanding, sorting, searching, and filtering data, including optional provenance attribute projections such as source path, source node, source output, source dataset, or source layer for reference imports. Bulk table editing is deferred so the table reinforces the procedural graph and layer model rather than becoming the primary editing surface.
